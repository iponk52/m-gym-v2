# Rencana Implementasi M-GYM (End-to-End GYM Management System)

Aplikasi ini merupakan sistem manajemen GYM lengkap dengan fitur registrasi member, digital member card (QR Code), sistem check-in/out, dan manajemen billing.

## Backend (Golang + Fiber + MySQL)
1. Inisialisasi Golang project.
2. Setup Koneksi Database (GORM + MySQL).
3. Buat Models (Member, Admin, Attendance, Subscription).
4. Buat Routes dan Handlers.

## Frontend (React + Vite + TailwindCSS)
1. Inisialisasi Vite + React.
2. Setup Pages (Dashboard, Members, Billing, Scanner).
3. Integrasi API.
